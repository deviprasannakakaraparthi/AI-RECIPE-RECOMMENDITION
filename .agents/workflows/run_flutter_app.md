---
description: Run Flutter frontend of AI Recipe Generator
---
1. Open terminal in the Flutter project directory (`frontend_flutter`).
   // turbo
2. Ensure Flutter SDK is installed and up to date.
   // turbo
3. Clean previous builds.
   // turbo
   ```
   flutter clean
   ```
4. Get dependencies.
   // turbo
   ```
   flutter pub get
   ```
5. Run the app in Chrome (web) for quick preview.
   // turbo
   ```
   flutter run -d chrome
   ```
6. If you have Android emulator set up, run on Android device.
   // turbo
   ```
   flutter run -d emulator-5554
   ```
